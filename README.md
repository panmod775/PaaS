# ☁️ PaaS - Nền tảng dịch vụ

Đây là repository chứa mã nguồn cho trang web tĩnh của dự án **PaaS**, được host trên GitHub Pages.

## 🌐 Địa chỉ truy cập
Bạn có thể truy cập trang web trực tiếp tại:
[https://panmod775.github.io/PaaS/](https://panmod775.github.io/PaaS/)


## 🛠️ Hướng dẫn cài đặt (dành cho nhà phát triển)
1. Clone repository:
   ```bash
   git clone https://github.com/panmod775/PaaS.git
   ```
2. Mở file `index.html` bằng trình duyệt để xem kết quả.

## 🤝 Đóng góp
Mọi ý kiến đóng góp hoặc báo lỗi xin vui lòng tạo [Issue](https://github.com/panmod775/PaaS/issues) hoặc gửi Pull Request.
